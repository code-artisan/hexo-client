# Hexo 客户端

## 依赖
需要保证有 `Node.JS` 运行环境和 `Git`。

## 开发
### 编译
```bash
$ npm start // 编译前后端资源，后端资源需重启后才生效
```

### 运行
```bash
$ electron .
```

## 打包
```bash
$ npm run pack:mas     # 各平台的打包命令请查看 package.json 文件
```
